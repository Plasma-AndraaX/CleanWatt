# CleanWatt

**Widget batterie Android grand format pour les personnes dont la vue baisse.**

Fini de plisser les yeux pour lire le minuscule indicateur de batterie en haut de l'écran. CleanWatt affiche le niveau en grand sur l'écran d'accueil : pourcentage, état de charge, température et santé. Mise à jour en temps réel, sans pub, sans tracking.

---

**Large Android battery widget for people with declining eyesight.**

Stop squinting at the tiny battery icon. CleanWatt displays your battery level in large format on your home screen: percentage, charging status, temperature, and health. Real-time updates, no ads, no tracking.

## Build

```bash
dotnet build -c Release
```
